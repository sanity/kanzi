3.2. Selecting target pages

Next you must select the pages to which you would like Kanzi to apply these changes. You can select each file individually using the "Add File(s)" button, or add all HTML files in a directory using the "Add a Directory" button.

You may now decide which of these files you would like those changes applied to, by clicking on them you may select and deselect them. Only the selected files will be modified.

A much more convenient way to select the files you would like to modify is to use the Similarity Filter. You can activate it by clicking on the check box. You may now move the slider up and down, and files will be selected and deselected based on their similarity to the example file which you modified in the previous step.

Click "Next" when the files you wish to modify are highlighted.
