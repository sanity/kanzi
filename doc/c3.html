Chapter 1. Introduction

This is the User Manual for Kanzi, Cematics' intelligent web development tool. Kanzi is designed to bring the flexibility of tools that computer programmers have been using for years, to the world of web development. Kanzi's core goal is to integrate seamlessly with the web development tools with which you are already familiar, meaning that you can get started with it immediately.

Kanzi is useful when you need to make changes to a large number of web pages. You demonstrate the changes you need made on one of the pages using your favourite HTML editing tool, and Kanzi will intelligently figure out what has changed, and how those changes are to be applied to the other pages you have selected.
