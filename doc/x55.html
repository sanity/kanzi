3.3. Reviewing changes

The changes will now be applied to the pages. This should occur quite quickly, but the time required may vary depending on the extent of the changes to the example page, and the number of target pages.

You may now review the changes Kanzi has made to these pages. You can launch a web-browser pointing at the (modified) example page by clicking on "Review Your Documents". You should check the target files to ensure that Kanzi has applied the changes correctly.

If it has, you can apply the changes permenantly by clicking on "Finish". You can undo the changes by clicking on "Undo".
