Chapter 2. Installation and Registration

2.1. Requirements

Kanzi requires Java version 1.2 or greater. This may be downloaded from http://java.sun.com/getjava, and will require just a couple of minutes to install for most users. Kanzi will work on any version of Windows that supports Java, including Windows 98, Windows 2000, Windows NT, Windows XP, and Windows ME.
