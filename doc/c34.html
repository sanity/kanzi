Chapter 3. Using Kanzi

There are three basic steps to using Kanzi:

  • Select the "example" file, and make the changes you want to it

  • Select the files to which you want those changes to be applied

  • Review the changes Kanzi made, then either commit or undo them

3.1. Training Kanzi

When you first start Kanzi, you will be invited to select the "example" file. This should be one of the HTML pages that you want to modify. You should then use your favourite HTML editor to make the modifications to that page that you want to apply to the other pages, and save it.

Once you have saved the file, you will be able to click on "Next" and move on to the next step.
